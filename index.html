# ChatGPT처럼 유니티 게임 개발

이 프로젝트는 생성형 AI 기술이 게임 개발 과정에 얼마나 혁신적으로 개입할 수 있는지 실증적으로 보여주는 것을 목표로 합니다. 단순한 스크립트 생성을 넘어, 사용자의 아이디어를 구조화된 데이터와 유니티 컴포넌트로 즉시 변환하여 게임 프로토타입을 제작하는 전 과정을 자동화하고자 합니다.

주요 목표

  • 자연어 프롬프트를 정확한 Unity C# 스크립트로 변환하는 AI 파이프라인 구축
  • Unity 에디터 내에서 AI 기능을 사용할 수 있는 커스텀 UI/UX 개발
  • AI 기술의 잠재력과 구현 가능성 입증

시연영상

실제 동작 흐름(프롬프트 입력 → 생성/적용 → 실행)을 녹화한 데모입니다.

영상에서 확인할 것

  • 입력: 자연어 프롬프트로 기능 요청
  • 분석: 요구사항을 단계별 구조(스키마)로 정리
  • 생성: Unity C# 코드/템플릿 생성
  • 적용: 씬 오브젝트 생성 + 컴포넌트 부착
  • 결과: 바로 Play로 프로토타입 확인
TIP: GitHub Pages에서 영상이 안 뜨면 파일 경로(assets/demo.mp4)와 대소문자를 확인하세요.

핵심 아키텍처

프로젝트의 핵심은 사용자 프롬프트가 실제 Unity 씬 구성 요소로 변환되는 워크플로우에 있습니다. 각 단계를 클릭하여 AI의 역할과 데이터 흐름을 확인할 수 있습니다.

P

Prompt 입력

A
B
C

Unity Game 실행

프롬프트 입력

사용자가 원하는 게임 아이디어를 자연어(예: "토끼가 당근을 모으는 2D 플랫포머")로 입력합니다. 이 입력이 전체 프로세스의 시발점입니다.

A. 프롬프트 분석 및 JSON 구조화 (LLM)

LLM이 프롬프트를 분석하여 게임 타입, 핵심 메커니즘, 필요한 에셋/스크립트를 포함하는 JSON 스키마로 변환합니다.

JSON 스키마 예시:

{
  "gameType": "2D Platformer",
  "mechanics": [
    {"name": "PlayerMovement", "scriptName": "PlayerController.cs"},
    {"name": "Collectable", "scriptName": "Collectable.cs"}
  ]
}

B. C# 스크립트 생성 (LLM)

JSON의 mechanics 목록을 기반으로 Unity의 MonoBehaviour를 상속하는 C# 스크립트를 생성합니다.

스크립트 요구사항:

  • Unity 표준 함수(Start/Update) 구현
  • 정확한 문법 및 Unity API 사용
  • 필요 변수(public) 및 주석 포함

C. Unity 통합 프레임워크 (개발자 영역)

AI가 생성한 C# 파일을 프로젝트 폴더에 저장하고, Unity 씬에 필요한 오브젝트를 생성한 뒤, 해당 오브젝트에 스크립트를 AddComponent로 자동 부착합니다.

주요 기능:

  • 스크립트 파일 시스템 통합
  • 씬 오브젝트 자동 생성 및 컴포넌트 부착
  • 에러 로그 기반 피드백(디버그 탭)

Unity Game 실행

모든 스크립트와 씬 구성이 완료되면, Unity 에디터에서 즉시 프로토타입 게임을 실행하여 결과를 확인합니다.

기술 스택

Unity / 클라이언트

  • Unity Editor 확장 (EditorWindow 기반 UI)
  • C# 자동 생성 및 파일 저장/컴파일 흐름
  • 씬 오브젝트 자동 생성 + 컴포넌트 부착(AddComponent)
  • 결과 확인을 위한 즉시 실행(Play) 프로세스

AI / 서버 & 데이터

  • LLM API 기반 프롬프트 엔진(분석 → 구조화 → 생성)
  • JSON 스키마를 통한 안정적인 데이터 전달
  • 기획 문서/코드 조각/로그/CSV 등 개발 데이터 연동(확장 방향)
  • GitHub Pages로 포트폴리오 배포

주요 기능(발표자료 기반)

기획 & 콘텐츠 보조
  • 세계관/스토리 초안 생성
  • 퀘스트/이벤트 아이디어 제안
  • 게임 기획서 초안 자동 작성
프로그래밍 & 디버깅
  • 에러 로그 분석 및 원인 추정
  • 자주 쓰는 코드 템플릿 생성
  • 코드 개선(improvement) 제안
데이터 밸런싱 자동화
  • 스탯/테이블 분석 및 조정안 제시
  • 아이템/스킬 CSV·JSON 템플릿 생성
  • 입력 오류 감소 & 일관성 유지
